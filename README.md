# Enterprise Architect Self-Assessment Tool

A self-assessment tool for enterprise architects, based on the [INNOCOM Institute](https://innocom.eu) competence model.

**Live tool:** https://christophewindelen.github.io/ea-self-assessment

Companion to the article series *[Prove It](https://medium.com/@christophe.windelen/you-say-you-are-an-architect-of-the-enterprise-prove-it-d785515d8e42)* on Medium.

---

## What it does

Walks through all five INNOCOM capability clusters, collects two scores per cluster (current capability and development investment), then generates a personalised report with a radar chart, gap analysis, 90-day action plan, and an if-then implementation planner.
