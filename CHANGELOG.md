# Changelog

## v1.0 — 2026-05-08

Initial release.

- Five INNOCOM capability clusters: Trusted Advisor, Strategic Ambition, Architectural Focus, Execution and Delivery, Solution Design
- Two scores per cluster: current capability (1 Emerging / 2 Developing / 3 Leading) and development investment (1 Minimal / 2 Moderate / 3 Active)
- Radar chart comparing both dimensions across all five clusters
- Gap analysis with status tags (Underinvesting / Low investment / Aligned / Growing into)
- Personalised insight paragraph based on score pattern
- 90-day action plan for top gaps
- If-then implementation planner (Gollwitzer and Sheeran, 2006)
- Calibration prompts to reduce self-assessment inflation
- CTA linking to INNOCOM Institute for structured assessment
